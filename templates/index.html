UMG Parser

How to Use

  1. Open any UMG Widget Blueprint in UEFN
  2. Select which widgets you want to convert from the hierarchy
  3. Copy to clipboard (Ctrl + C)
  4. Paste (Ctrl + V) into the input box on the right
  5. Don't forget to put Verse Utils anywhere in your code

Declaring Variables

Prefix any widget with $ to create a Verse variable.

The converter will declare it at the top.

Example: $MyButton

Placeholder Widgets

Use __ignore anywhere in the name to skip converting that widget.

External Widgets

Prefix with external_ to reference widgets outside the hierarchy.

Example: external_Player.Widget

This generates:
Widget := Player.Widget

Auto Generated Messages

When enabled, all localized texts generate Verse messages automatically.

Requires proper localization key names in UEFN.

If disabled, raw text is used with .Msg().

Supported Widgets

  • Canvas / CanvasSlot
  • StackBox / StackBoxSlot
  • Overlay / OverlaySlot
  • Text Block
  • Image Block / Material Block / Color Block
  • Buttons
  • Slider
